ShopStream

Real-Time Data Pipeline for E-Commerce Analytics

GitHub Repo: ShopStream on GitHub

📌 Project Overview

ShopStream is a real-time data pipeline project designed for e-commerce analytics using a minimal tech stack. It simulates data ingestion, transformation, storage, and visualization end-to-end.

⚙️ Tech Stack

  • Ingestion: Python (Simulated Stream), Kafka (optional)
  • Processing: PySpark / Pandas
  • Storage: Azure Blob / AWS S3 / CSV
  • Data Warehouse: SQLite / Synapse / Redshift
  • Visualization: Power BI / Matplotlib
  • Orchestration: Python (or Airflow)
